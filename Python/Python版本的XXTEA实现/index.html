Python版本的XXTEA实现

关于xxtea

  • xxtea一种简单、执行快速的加密方式，通常很少的代码量就能实现，缺点就是安全性不高
  • 对称密钥加密的一种：加密解密使用相同的密钥
  • 全称：Tiny Encryption Algorithm（中文：微型加密算法），前身是xtea；xtea的前身又是tea，最新的加强版本就是xxtea了
